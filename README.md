### Some documentations...
The directory structure looks like this:<br><pre>
AndroidManifest.xml<br>
src/<br>
res/<br>
assets/<br>
  boot-animation/<br>
  fonts/<br>
  icons/<br>
    res/ (note: same dir structure as res/ in trebuchet icon packs)<br>
  wallpapers/<br>
  ringtones/<br>
  notifications/<br>
  alarms/<br>
  overlays/<br>
      [target-pkg-name1]<br>
          res/<br>
           drawable/<br>
           drawable-hdpi/<br>
           ...<br>
           layouts/<br>
           xml/<br>
      [target-pkg-name2]<br>
          res/</pre><br>
